Bài 17: Kiểm tra số chia hết cho 3

Viết chương trình kiểm tra một số người dùng nhập có chia hết cho 3 không.

n = int(input("Nhập một số: ")) if n % 3 == 0: print(f"{n} chia hết cho 3") else: print(f"{n} không chia hết cho 3")
# Cách 1: Sử dụng toán học - tổng các chữ số chia hết cho 3 n = input("Nhập một số: ") digit_sum = sum(int(digit) for digit in n) if digit_sum % 3 == 0: print(f"{n} chia hết cho 3") else: print(f"{n} không chia hết cho 3")
# Cách 2: Sử dụng hàm để kiểm tra def is_divisible_by_3(num): return num % 3 == 0 n = int(input("Nhập một số: ")) if is_divisible_by_3(n): print(f"{n} chia hết cho 3") else: print(f"{n} không chia hết cho 3")
# Cách 3: Sử dụng phép chia nguyên và phần dư n = int(input("Nhập một số: ")) quotient = n // 3 remainder = n - (quotient * 3) if remainder == 0: print(f"{n} chia hết cho 3") print(f"{n} ÷ 3 = {quotient}") else: print(f"{n} không chia hết cho 3") print(f"{n} ÷ 3 = {quotient} dư {remainder}")

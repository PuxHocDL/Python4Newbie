Bài 19: Tạo chuỗi lặp lại

Viết chương trình lặp lại một chuỗi người dùng nhập đúng n lần (n cũng do người dùng nhập).

s = input("Nhập chuỗi: ") n = int(input("Nhập số lần lặp: ")) print(s * n)
# Cách 1: Sử dụng vòng lặp for s = input("Nhập chuỗi: ") n = int(input("Nhập số lần lặp: ")) result = "" for i in range(n): result += s print(result)
# Cách 2: Sử dụng hàm repeat và join def repeat_string(string, times): return "".join([string] * times) s = input("Nhập chuỗi: ") n = int(input("Nhập số lần lặp: ")) print(repeat_string(s, n))
# Cách 3: Sử dụng list comprehension và join s = input("Nhập chuỗi: ") n = int(input("Nhập số lần lặp: ")) result = "".join([s for _ in range(n)]) print(result)

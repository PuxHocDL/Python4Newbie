Bài 18: Tính chu vi hình chữ nhật

Viết hàm tính chu vi hình chữ nhật từ chiều dài và chiều rộng người dùng nhập.

def rectangle_perimeter(length, width): return 2 * (length + width) length = float(input("Nhập chiều dài: ")) width = float(input("Nhập chiều rộng: ")) print("Chu vi hình chữ nhật:", rectangle_perimeter(length, width))
# Cách 1: Sử dụng lambda function rectangle_perimeter = lambda l, w: 2 * (l + w) length = float(input("Nhập chiều dài: ")) width = float(input("Nhập chiều rộng: ")) print("Chu vi hình chữ nhật:", rectangle_perimeter(length, width))
# Cách 2: Tính trực tiếp không dùng hàm length = float(input("Nhập chiều dài: ")) width = float(input("Nhập chiều rộng: ")) perimeter = 2 * (length + width) print("Chu vi hình chữ nhật:", perimeter)
# Cách 3: Tạo lớp Rectangle class Rectangle: def __init__(self, length, width): self.length = length self.width = width def perimeter(self): return 2 * (self.length + self.width) length = float(input("Nhập chiều dài: ")) width = float(input("Nhập chiều rộng: ")) rect = Rectangle(length, width) print("Chu vi hình chữ nhật:", rect.perimeter())

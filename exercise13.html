Bài 13: Kiểm tra chuỗi đối xứng

Viết chương trình kiểm tra chuỗi người dùng nhập có phải chuỗi đối xứng không (ví dụ: "radar" là chuỗi đối xứng).

s = input("Nhập chuỗi: ") if s == s[::-1]: print("Đây là chuỗi đối xứng") else: print("Đây không phải chuỗi đối xứng")
# Cách 1: Sử dụng vòng lặp để so sánh từng ký tự s = input("Nhập chuỗi: ") is_palindrome = True for i in range(len(s) // 2): if s[i] != s[len(s) - i - 1]: is_palindrome = False break if is_palindrome: print("Đây là chuỗi đối xứng") else: print("Đây không phải chuỗi đối xứng")
# Cách 2: Sử dụng hàm reversed() s = input("Nhập chuỗi: ") reversed_s = "".join(reversed(s)) if s == reversed_s: print("Đây là chuỗi đối xứng") else: print("Đây không phải chuỗi đối xứng")
# Cách 3: Sử dụng hàm để kiểm tra def is_palindrome(s): return s == s[::-1] s = input("Nhập chuỗi: ") if is_palindrome(s): print("Đây là chuỗi đối xứng") else: print("Đây không phải chuỗi đối xứng")

Bài 10: Tính diện tích hình tròn

Viết hàm tính diện tích hình tròn từ bán kính.

def circle_area(r): return 3.14 * r * r r = float(input("Nhập bán kính: ")) print(f"Diện tích hình tròn: {circle_area(r):.2f}")
# Cách 1: Sử dụng hằng số pi từ thư viện math import math def circle_area(r): return math.pi * r ** 2 r = float(input("Nhập bán kính: ")) print(f"Diện tích hình tròn: {circle_area(r):.2f}")
# Cách 2: Sử dụng lambda function import math circle_area = lambda r: math.pi * r * r r = float(input("Nhập bán kính: ")) print(f"Diện tích hình tròn: {circle_area(r):.2f}")
# Cách 3: Tính trực tiếp không dùng hàm import math r = float(input("Nhập bán kính: ")) area = math.pi * r * r print(f"Diện tích hình tròn: {area:.2f}")

Bài 16: Tạo bảng cửu chương

Viết chương trình in bảng cửu chương của một số người dùng nhập (từ 1 đến 10).

n = int(input("Nhập số để in bảng cửu chương: ")) for i in range(1, 11): print(f"{n} x {i} = {n * i}")
# Cách 1: Sử dụng vòng lặp while n = int(input("Nhập số để in bảng cửu chương: ")) i = 1 while i <= 10: print(f"{n} x {i} = {n * i}") i += 1
# Cách 2: Sử dụng list comprehension và join n = int(input("Nhập số để in bảng cửu chương: ")) table = [f"{n} x {i} = {n * i}" for i in range(1, 11)] print("\n".join(table))
# Cách 3: Tạo một hàm và dùng map def multiply(num, multiplier): return f"{num} x {multiplier} = {num * multiplier}" n = int(input("Nhập số để in bảng cửu chương: ")) results = map(lambda i: multiply(n, i), range(1, 11)) for result in results: print(result)

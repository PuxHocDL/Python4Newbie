Bài 20: Tìm số lớn thứ hai

Viết chương trình tìm số lớn thứ hai trong danh sách 5 số người dùng nhập.

lst = [] for i in range(5): num = int(input(f"Nhập số thứ {i+1}: ")) lst.append(num) lst.sort() print("Số lớn thứ hai là:", lst[-2])
# Cách 1: Sử dụng hàm max() hai lần lst = [] for i in range(5): num = int(input(f"Nhập số thứ {i+1}: ")) lst.append(num) max_num = max(lst) lst.remove(max_num) second_max = max(lst) print("Số lớn thứ hai là:", second_max)
# Cách 2: Không sử dụng hàm có sẵn lst = [] for i in range(5): num = int(input(f"Nhập số thứ {i+1}: ")) lst.append(num) max_num = float('-inf') second_max = float('-inf') for num in lst: if num > max_num: second_max = max_num max_num = num elif num > second_max and num != max_num: second_max = num print("Số lớn thứ hai là:", second_max)
# Cách 3: Sử dụng set để loại bỏ trùng lặp lst = [] for i in range(5): num = int(input(f"Nhập số thứ {i+1}: ")) lst.append(num) # Chuyển sang set để loại bỏ trùng lặp, rồi chuyển lại thành list và sắp xếp unique_lst = list(set(lst)) unique_lst.sort() if len(unique_lst) >= 2: print("Số lớn thứ hai là:", unique_lst[-2]) else: print("Không có số lớn thứ hai (tất cả số đều giống nhau)")
